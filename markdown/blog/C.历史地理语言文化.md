# 雅威、蝴蝶和避讳

关于日语“蝶”的训读消失的问题。这是避讳和敏感词影响语言演化的一个生动实例。类似的例子还有犹太四字神名YHWH（雅威，即“耶和华”），历史上由于避讳神名而丢失了元音。语言文字会塑造人的思维，人的思维反过来也会塑造语言文字。（2021-11-27）

- **#(耶和华)#**或**#(雅威)#**：希伯来古神名“יהוה”，拉丁化为“YHVH”。由于犹太人为避讳而不直呼其名，改为称呼Adonai“神”之类的代称，再加上文字体系并不强调元音，导致其元音失传，无法考证。中文一般翻译为“耶和华”或者“雅威”。意义可能是“自有永有者”。（见维基百科“四字神名”等词条）
- 日语的“蝴蝶”，只有音读，没有训读。据[资料](https://zhuanlan.zhihu.com/p/34343116)，日语“蝴蝶”的固有词，即“蝶”的训读（可能是かわひらこ）已经丢失，原因大概是因为大和人认为蝴蝶是亡灵的化身，因而在和歌等文献中避讳“蝴蝶”这个词的固有说法，导致训读消失，仅剩汉语音读。其他参考资料：[1](https://superokan.hatenablog.com/entry/2016/12/08/063645)、[2](https://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q118837526)

# 瞳孔和小孩

pupil和「小狗」puppy、「傀儡」puppet也是同源词，它们的本义不是小男孩就是小女孩。表示小男孩的pupus最终衍生出了pupil，而它的阴性即表示小女孩的pupa，则衍生出了玩偶的意思。除了汉语和英语，日语「瞳」训读也可以写成「人見」，和汉字「瞳」有着异曲同工之妙。据说古巴比伦也有过「住在眼中的守护神」这样的说法，守护神很可能指的就是瞳孔中的小人。

# 汉语探源

## 文白分离与言文关系

以下是2025-10-09与Qwen3-Max的对话记录。

**提问**：

“子孙永宝用”是什么意思？

**Qwen3-Max**：

“子孙永宝用”是一句常见于中国古代青铜器铭文中的吉语，意思是：希望子孙后代永远珍视并使用这件器物。

具体解释如下：子孙：指后代子孙。永：永远、长久。宝：作动词，意为“珍视”“视为宝物”。用：使用。

整句话表达了器物主人对家族传承的美好愿望，希望这件器物（通常是礼器、祭器或贵重器物）能够被后代子孙世代珍藏、使用，象征家族的延续、福泽的绵长，也体现了古人重视宗族传承和器物礼制的文化观念。

这类铭文多见于商周时期的青铜器上，如鼎、簋、盘等，常用于祭祀或重要场合，具有宗教、礼仪和纪念意义。类似的铭文还有“子子孙孙永宝用享”“万年永宝”等。

------

**提问**：

那个时代的汉语有哪些特征？与现代汉语有哪些区别？

**Qwen3-Max**（摘要）：

“那个时代”如果指的是“子孙永宝用”这类铭文所处的时代，即商周时期（约公元前1600年—前256年），尤其是西周至春秋战国阶段，那么当时的汉语属于**上古汉语**（Old Chinese）。这是汉语发展的最早阶段之一，与现代汉语（特别是普通话）在语音、词汇、语法等方面存在显著差异。下面简要说明上古汉语的主要特征及其与现代汉语的区别：

| 方面         | 上古汉语（商周至战国）           | 现代汉语（普通话）               |
|---------------------------------------------------------------------------|
| 音节结构     | 单音节为主，可能有复辅音         | 双音节为主，无复辅音             |
| 声调         | 无（或未分化）                  | 有四声                        |
| 词汇         | 单音词为主，通假普遍             | 双音词为主，词汇丰富、抽象化       |
| 语法         | 宾语可前置，虚词用法不同          | 语序固定，依赖“的、了、着”等助词   |
| 书写系统     | 甲骨文、金文、大篆，未统一        | 简化字/繁体字，高度规范化         |
| 代表文献     | 《诗经》《尚书》《金文铭辞》等     | 现代白话文作品                   |

------

**提问**：

汉语文白分离是从什么时候开始的？原因是什么？为什么近代言文一致了？

**Qwen3-Max**（摘要）：

汉语文白分离（即书面语与口语严重脱节）是一个长期演变的过程，其形成、持续与最终消解涉及语言、社会、政治、教育等多重因素。以下是对你三个问题的系统解答：

一、文白分离是从什么时候开始的？

文白分离的萌芽可追溯至先秦，但真正形成并固化是在汉代以后，尤其在魏晋至唐宋时期趋于明显。

- 先秦时期（上古）：书面语（如《论语》《孟子》《左传》）虽已较口语简练、典雅，但与当时口语差距尚不悬殊。例如《论语》中“学而时习之，不亦说乎？”仍较接近当时的口语表达。
- 两汉时期（中古早期）：随着儒家经典被奉为“圣言”，以《五经》为代表的先秦书面语被神圣化、经典化。人们写作时模仿先秦文体，而日常口语却在自然演变，书面语开始“复古”而口语持续“更新”，文白差异初现。
- 魏晋南北朝至唐宋：文人普遍以“古文”（先秦两汉文体）为正统，科举考试、官方文书、学术著述皆用文言。与此同时，口语已发展出中古汉语的特征（如声调系统成熟、词汇变化）。文言成为固定不变的书面规范，口语则不断变化，二者渐行渐远，文白分离正式形成。
- 元明清时期：文言彻底成为“死语言”（不再随口语变化），而白话（以北方话为基础的口语）在戏曲、小说（如《水浒传》《红楼梦》）中广泛使用，形成“文言用于正式场合，白话用于通俗文学”的双轨制。

结论：文白分离在汉代开始萌芽，魏晋唐宋时期确立，宋元以后完全固化。

| 阶段         | 特征                     | 关键动因                     |
|----------------------------------------------------------------------|
| 先秦         | 文白基本一致             | 书面语源于口语               |
| 汉至唐宋     | 文白开始分离并固化       | 经典神圣化、科举制度         |
| 元明清       | 文白严重脱节             | 文言为士人专属，白话用于俗文学 |
| 晚清至五四   | 推动言文一致             | 启蒙、救亡、教育、媒体变革   |
| 1920年后     | 白话文成为标准书面语     | 国家政策、新文化运动成功     |

如今，现代汉语书面语虽仍保留少量文言成分（如成语、公文用语），但整体已实现“我手写我口”的言文一致，这是中国语言现代化的重要标志。

**提问**：

作为一个现代人，先秦口语与书面语一致，让我很难理解。有这样一种观点，认为从甲骨文开始的各类文献，实际上并没有完全记录下当时汉语的丰富的词缀和屈折变化现象（而是用不同的字来记录，例如“登”与“增”实际上是同一个词的不同屈折形式），犹如阿拉伯文一般并不记录元音，并认为这就是古代汉语言文分离的一个开端。请运用专业的语言学知识，分析这种观点。

**Qwen3-Max**（摘要）：

这是一个极具挑战性且在语言学界存在激烈争论的问题。你提到的观点——**先秦汉语可能存在未被文字系统充分记录的形态变化（如词缀、屈折），而甲骨文、金文等早期汉字系统因“不记录语音细节”（类似阿拉伯文省略元音）导致书面语无法反映口语的真实形态，从而构成文白分离的早期源头**——确实有部分学者（如沙加尔 Laurent Sagart、白一平 William Baxter、郑张尚芳等）在重构上古汉语时提出过类似假设。但这一观点需要谨慎对待，下面从专业语言学角度进行系统分析。

一、核心主张的合理性：汉字是否“掩盖”了上古汉语的形态？

1. 汉字的本质：语素文字，非拼音文字。

汉字是语素-音节文字（morphosyllabic script），每个字通常对应一个语素（意义单位）和一个音节。它不直接记录音素细节（如辅音丛、元音长短、声调、词缀音变等），尤其在早期（甲骨文、金文）字形尚未完全定型时，存在大量假借、通假现象。例如：“其”本义为“簸箕”，但常假借为代词或语气词；“来”本义为“麦子”，假借为动词“到来”。这种一字多用、音近通假的现象，使得我们难以从字形直接推断当时的实际发音或语法形态。

2. 上古汉语可能存在形态变化的证据

尽管传统观点认为汉语自古就是“孤立语”（analytic language，无屈折变化），但近几十年的上古汉语研究提出：

- 前缀/后缀假说：沙加尔（Sagart）等人提出上古汉语存在派生性词缀，用于表达使动、名物化、反义等语法功能。
- 声母交替与形态相关：一些字对（如“见”“现”）显示清浊交替可能与主动/被动、自动/使动有关。
- 汉藏语比较的支持：藏语、缅语等亲属语言保留丰富的屈折和派生形态，暗示原始汉藏语可能有类似系统，汉语后来简化了。

结论：上古汉语很可能存在一定程度的形态变化（主要是派生，而非屈折），但这些变化未被汉字系统忠实记录，因为汉字重“义”轻“音”，且字形与语音的对应关系松散。

二、类比“阿拉伯文不记元音”是否恰当？

这个类比部分成立，但有重要区别：

- 相似点：两者都未完整记录语音细节，导致现代人难以直接从文字还原口语形态。
- 关键区别：阿拉伯文的省略是系统性规则（所有文本都省元音），而汉字的问题在于一个字可能对应多个音或语法形式，且通假、假借使得字与词的关系模糊。

因此，说“汉字像阿拉伯文一样不记录形态”是一种启发式比喻，有助于理解文字对语言的“遮蔽”，但技术上不精确。

三、这是否构成“文白分离的开端”？

不能简单等同。原因如下：

- 文白分离的核心是“书面语主动模仿古代文体”，而非“文字无法记录口语”。即使上古口语有形态变化，当时的书面语（如《诗经》《尚书》）仍被认为是对当时口语的较忠实记录（尽管经过文学加工）。真正的文白分离始于汉代以后士人刻意模仿先秦文体，而非因为甲骨文“没写全”导致脱节。
- 早期文献仍反映口语特征。甲骨卜辞（如“王大令众人曰协田”）句式简短，接近口语；《论语》中大量对话体（“子曰：‘学而时习之……’”）被认为保留了春秋时期口语风格。
- 文字局限 ≠ 文白分离。所有古代文字系统（如苏美尔楔形文字、埃及圣书体）都有记录局限，但并非都导致文白分离。文白分离是社会文化选择（崇古、科举、精英垄断）的结果，而非单纯技术限制。

更准确的说法：汉字系统对语音和形态的“不透明性”，为后来文言的固化提供了条件（因为后人无法准确还原先秦发音，只能模仿字面），但它本身不是文白分离的直接原因，更不是“开端”。

这一问题触及汉语语言学最前沿的争议，也提醒我们：文字只是语言的影子，而非语言本身。

**提问**：

你提到“来”本义为“麦子”，假借为动词“到来”。请详细解释。

**Qwen3-Max**（摘要）：

这是一个非常经典的汉字假借案例，体现了汉字发展过程中“本义”与“假借义”的分离现象。甲骨文“来”与“麦”字密切相关。事实上，“麦”字是在“来”的基础上加“夊”（表示脚，象征“外来”）构成的，说明“麦”是后起字。在商代，“来”专指麦类作物（可能特指小麦）。例证：甲骨卜辞中有“受来年”（意为“获得麦子的丰收年”），这里的“来”明显指农作物。《诗经·周颂·思文》：“贻我来牟” —— “来牟”即“麦”（“牟”为大麦，“来”为小麦），郑玄注：“来，小麦；牟，大麦也。”所以，“来”的本义是麦子，尤其指小麦。

“来”假借为“到来”，是语音驱动的借用。上古汉语中，表示“到来”这个动作的词，其发音与“麦子”的“来”非常接近甚至相同。“麦子”和“到来”在上古是同音词或音近词。然而，商周时期，汉字数量有限，遇到有音无字的词时，常借用同音或音近的已有字来记录。“到来”这个动词当时可能没有专用字，于是借用了发音相同的“来”（麦子）字来书写。“到来”是高频动词，使用频率远高于“麦子”这一名词。随着时间推移，人们看到“来”字，首先想到的是“到来”，而非“麦子”。为避免混淆，后来另造“麦”字（从“来”+“夊”，“夊”表示脚，暗示麦子是“从外方传入的作物”）来专指麦子。“麦”字出现后，“来”的本义逐渐被“麦”取代，“来”则专用于假借义“到来”。

| 时期       | “来”字用途                     | 相关字形/文献证据                     |
|---------------------------------------------------------------------------------|
| 商代甲骨文 | 本义：麦子                    | 卜辞“受来年”；字形象麦穗             |
| 西周金文   | 仍可指麦子，但假借“到来”已常见 | 《大盂鼎》等铭文中“来”多作动词       |
| 春秋战国   | 假借义“到来”成为主流           | 《论语》《孟子》中“来”几乎全为动词   |
| 汉代       | 本义完全由“麦”承担             | 《说文解字》：“来，周所受瑞麦……假借为行来之来” |

许慎在《说文解字》中明确指出：“来，周所受瑞麦来麰也……天所来也，故为行来之来。”这句话的意思是：“来”本是周人所受的祥瑞麦子（来麰），因为麦子是“天赐而来”的，所以被假借为“行来之来”（即“到来”）。——这反映了汉代人对假借过程的理解（虽带神话色彩，但承认了假借事实）。

**提问**：

为什么上古汉语中，表示“到来”这个动作的词，其发音与“麦子”的“来”非常接近甚至相同。

**Qwen3-Max**（摘要）：

简言之：**“到来”与“麦子”在上古汉语中读音相近甚至相同，并非偶然，而是很可能源于同一个词根，通过“词义引申”或“文化隐喻”分化出两个看似无关的义项，而语音保持一致，最终导致文字假借。**

一、语音事实：二者在上古确为同音或极近音。二者拟音完全相同或仅在前缀上有微小差异**，说明在商周时期，这两个词语音上无法区分。这就为“假借”提供了基础：当人们需要书写“到来”这个动词时，自然会选择发音相同的“来”（麦子）字来记录。

二、词源学解释：是否同源？是否存在语义联系？“麦子”和“到来”是两个独立的词碰巧同音，还是本为一词，后因引申或文化隐喻分化？主流观点认为，两者是同源词，语义通过“外来之物”隐喻关联。这是更被广泛接受的解释，认为“来”（麦子）与“来”（到来）本为一词，核心语义是“从外部到来之物”。因为小麦并非中国本土原产作物。考古证据表明，小麦约在距今5000–4000年前（龙山文化至二里头时期）从中亚经河西走廊传入中原。对商周人而言，麦子是一种“外来的、从远方而来”的作物。“来”最初可能就是指“从远方传入的谷物（麦）”，其名称本身就包含“到来”之意，后来该词被抽象化，独立出“到来”这一动词用法。

三、语言类型学与跨语言证据。在世界语言中，“外来作物名称”常与“到来”“给予”“远方”等概念相关。例如：英语 “rice”（稻米）源自希腊语oryza，而oryza可能源于南印度语，意为“去壳的谷物”——强调其“加工后输入”。拉丁语 “frumentum”（谷物）与ferre（带来）同源。在一些语言中，“小麦”一词直接意为“来自西方的谷物”。这表明：将新引入的作物命名为“从某处来的东西”是一种跨文化的命名策略。汉语“来”正是这一策略的体现。

四、语音演变的佐证：为何后来未分化？如果“麦子”和“到来”是两个词，为何语音未随语义分化而产生差异？因为假借发生后，文字统一强化了语音同一性；汉代以后，“来”专用于“到来”，“麦”承担本义，但语音仍继承自同一源头，故中古音（《切韵》）中“来”（动词）与“麦”仍同属“咍韵”或相近韵部，显示同源痕迹。

总结：为何“到来”与“麦子”同音？根本原因在于：“来”最初指的是一种“从外部传入的谷物（小麦）”，其名称本身就蕴含“到来”之意。因此，“麦子”与“到来”并非两个独立词偶然同音，而是同一词根通过语义泛化产生的分化——前者保留具体指称，后者抽象为动词。语音的一致性正是同源关系的体现，而汉字假借只是这一语言事实的文字反映。

这一案例生动说明：上古汉语词汇并非孤立存在，而是与农业史、文化交流、认知隐喻紧密相连；汉字假借的背后，往往隐藏着深层的词源与文化逻辑。

延伸思考：类似现象还有：“年”：本义为“谷熟”（《说文》：“年，谷熟也”），因谷物一年一熟，引申为时间单位“年”；“叔”：本义为“拾取豆子”，后假借为兄弟排行。这些都体现了汉语词汇从具体农耕经验向抽象概念演化的路径。

**提问**：

基于你的回答，我继续提问。“来”是一个基础的、重要的动词，理应早已出现。动词“来”和名词“麦”在上古汉语中同音，是否可以这样理解：先有“来”的语音，后以“来”的语音命名“麦”这个概念。但是，对于文字的演化来说，“来”作为一个基础的抽象的词，理应很早出现其专用汉字，但为什么偏偏是语音上晚出现的“麦”词汇先获得了“来”符号，导致语音上先出现的“来”概念反而要借用这个“来”符号呢？

**Qwen3-Max**（摘要）：

这是一个极具洞察力的问题，触及了语言（口语）与文字（书写）发展不同步的核心矛盾。你的推理逻辑非常清晰，但其中隐含一个关键前提需要修正：**“基础动词理应更早拥有专用字”这一假设，在汉字起源的早期并不成立**。下面我将从语言学、文字发生学和考古证据三个层面，系统解答你的疑问。

一、核心澄清：语言先于文字，且早期文字只为“可象形/可指事”的概念造字。首先，动词“来”（到来）是抽象动作，难以象形。汉字起源以象形和指事为主，具体名词（如“人”“日”“山”“麦”）容易画出形状，但抽象动词（如“来”“去”“思”“爱”）无法直接描绘。因此，在文字初创阶段（甲骨文时代），抽象动词往往没有专用字，而是通过假借已有字来记录。而“麦”作为具体农作物，天然具备象形条件。小麦有鲜明的穗形，甲骨文“来”字清晰描绘了下垂的麦穗，这是典型的象形字，符合早期造字逻辑。所以，并非“麦”词汇晚出现，而是“麦”作为具体物象，比抽象动词“来”更容易获得专属字形。

二、语言与文字的时间差：“来”（动词）口语早于文字，“来”（麦）字形早于动词专用字。动词“来”口语出现极早，原始汉藏语时期已有“来”义动词，而名词“来”（麦子）口语出现晚于小麦传入。动词“来”的口语形式确实远早于小麦传入，但文字系统在商代才成熟，而此时小麦已成为重要作物。不是“麦”先有语音，而是“麦”先有可书写的形象。动词“来”虽口语早，但因抽象，无法造字，只能“寄生”于同音的具体名词字上。

三、为什么不用其他方式为“来”（动词）造字？既然“来”（动词）如此重要，为何不造一个专用字？因为早期汉字尚未发展出成熟的形声或会意机制来表示抽象动词。甲骨文中表示动作的字多靠假借（如“又”借为“有”，“其”借为代词），直到西周金文以后，才逐渐出现用“形旁+声旁”构造抽象词的形声字，而“来”字在商代已被固定用于记录“麦”，且假借为“到来”已成惯例，路径依赖使其难以替换。类似案例还有：“自”本义为“鼻子”（象形），假借为反身代词“自己”；“我”本义为一种兵器，假借为第一人称代词。这些都说明：早期汉字优先为具体物象造字，抽象概念靠假借。

四、你的假设为何看似合理但不符合历史事实？你提出“基础动词理应更早有专用字”，认为“来”“去”“吃”“看”是基础词汇，理应优先书写，但对商代人而言，祭祀、农业、战争才是文字的核心用途。甲骨文80%以上是占卜记录，关注“收成”“降雨”“征伐”，“麦子丰收”（受来年）比“某人到来”更需记录。因此，“麦”作为重要经济作物，优先获得字形。换言之：文字的创造由社会需求驱动，而非语言频率驱动。

五、后续发展：为何不另造字区分？实际上，“麦”字就是为区分而造的！这正是汉字“分化字”的典型策略，类似“其”（簸箕）假借为代词，后造“箕”专指簸箕；“莫”（日落）假借为否定词，后造“暮”专指傍晚。

六、总结：并非“麦”词汇晚出却抢了字形，而是“麦”作为具体物象，在文字诞生时恰好具备造字条件，而动词“来”只能暂时“借用”其字形。汉字是“为记录而生”的实用系统，而非“为语言建模”而设计的逻辑系统。它优先满足可写性（能否画出来）和社会急需（祭祀、农业、政令），而非语言学上的“基础性”。这也正是文言与口语逐渐脱节的深层原因之一。

- **文字≠语言**：文字系统在商代才成熟，而动词“来”的口语虽早，但因抽象无法象形；
- **造字优先级**：早期汉字只为可视觉化的概念造字，“麦子”符合这一条件；
- **假借是早期常态**：抽象词普遍借用同音具体词字，这是汉字发展的必经阶段；
- **社会需求导向**：农业词汇（如麦）在占卜文献中比抽象动词更需记录；
- **后续已分化**：“麦”字的创造正是为解决这一“借用冲突”。

**提问**：

那么，在“来”被假借为口语“来”的文字之前，表示动词“到来”含义的字符是什么？

**Qwen3-Max**（摘要）：

这是一个关键的问题，触及汉字早期发展中的“有音无字”状态。简明回答是：早期汉字数量有限，抽象动词普遍“无字可用”。在“来”字被假借用于记录动词“到来”之前，很可能没有专门的字符表示“到来”这个动词。人们要么用其他方式迂回表达，要么在书写中暂时空缺，直到借用“来”字成为惯例。

在早期甲骨文（武丁时期）中，“来”几乎全指麦子或贡纳之物（如“羌来”可能指羌人进贡麦子或物资）。而晚期甲骨文（祖庚、祖甲以后）出现“王来”“子来”等用法，明显表“某人到来”。这说明：“来”字从“麦子”义向“到来”义的假借，是在商代中晚期逐步发生的。

比较语言学显示，原始汉藏语已有表示“来”的动词词根，这说明“来”作为动词在汉语分化前就已存在，远早于小麦传入和汉字诞生。

但口语存在 ≠ 文字存在。在“来”字被假借之前，汉语动词“到来”没有专用字符。人们或依赖上下文省略，或用“至”“入”等近义词替代，或在必须书写时暂时无法表达。直到商代中晚期，因“来”（麦）与“来”（到来）同音，才开始借用“来”字记录该动词，并最终固定下来。这一过程体现了汉字发展的基本规律：**具体名词优先造字 → 抽象词靠同音假借 → 后期通过分化字（如“麦”）解决混淆**。

**提问**：

今年是2025年，网络语言、网络段子的代谢极其活跃，例如日语“蜂蜜”音译为“哈基米”通过互联网和动漫渠道传入中文互联网后发生变异，演变成一种娱乐化的对于“猫”的指称。针对上面提到的“具体名词优先造字 → 抽象词靠同音假借 → 后期通过分化字（如“麦”）解决混淆”这样的语言现象，你能举出一个当代网络语言网络段子能够反映这种语言现象的案例吗？

**Qwen3-Max**（摘要）：

案例：“蚌埠住了” → “绷不住了” → “🦪”作为情绪崩溃的符号化表达

第一阶段：具体名词（“蚌埠”）作为语音载体被借用（≈ 上古“来”=麦子）。此时，“蚌埠”作为具体地名（相当于“麦子”），被假借来记录一个抽象心理状态，完全依赖语音相同/相近。
第二阶段：假借义迅速流行并脱离原义（≈ “来”字主要用于“到来”）。“蚌埠”在此语境中完全符号化，仅作为“bèng bù”音节的视觉载体。如同“来”字本义“麦子”被遗忘，“蚌埠”的地名义在该用法中被悬置。
第三阶段：出现“分化”与“再抽象”趋势（≈ 造“麦”字区分本义）。为避免混淆，部分网友开始用“🦪”代表“绷不住了”。

这相当于通过**缩略、符号化、语义剥离**，实现“假借义”与“本义”的**功能性分化**——“🦪”成为独立的情绪符号，而“蚌埠”仍保留地名功能。


## 复辅音

2025-01-02

有人说现在主流的PST（原始汉藏语）乃至上古汉语的构拟都有大量的复辅音，说起来很拗口；加上B站大量的朗读视频给人一种嘴里含着西梅说话的感觉，让很多人对构拟结果抱有深深的怀疑。但也有人认为实际上富含复辅音的语言说起来是十分流畅甚至好听的。因为实际说话时会有很多音变规则和发语词一类的东西。比如罗刹囯之所以叫俄罗斯，是因为蒙古语似乎不允许r开头。

PST并不是汉语发展的起点，高度的屈折和复辅音化也只是语言螺旋发展的一个阶段，只是我们恰好只能追溯到那个阶段。有人认为，语言的高度开音节化，就是复辅音化的开端。例如日语表示现在时的后缀“ている”似乎有丢掉元音i变成“てる”的趋势。一些网络meme神秘缩写实际上也反映出这样的趋势，比如“yyds”一类的，这正是有些闪-含语言的标记方式——只标记辅音。久而久之甚至可能反馈回口语：元音就不重要了。四字神名YHVH就是一个案例。起初大家只是玩抽象，结果玩着玩着真的抽象起来了。

至于为什么语言发展倾向于保留辅音而丢掉元音，我不负责任地猜想，因为辅音的频谱相对比较白，频谱利用率更高，单位时间内能传达更多的符号；而元音的频谱呈现出一系列的共振峰，频谱利用率低，更多地起到一种类似于数字通信中保护间隔（以防止符号间干扰ISI）的功能，以及某种载波的功能。古代汉语的复辅音前后缀逐渐演变成如今普通话的四声（以及粤方言的n声），有种高频率的单载波模式像低频率的多载波模式演进的意思。

## 上古汉语二源论

之前见过一个很有趣（但未必正确）的观点：上古汉语有两个主要源头，一个是炎黄（仰韶）的羌语（华夏语），另一个是蚩尤（红山）的东夷语。甲骨文原本是记录东夷语的文字，后来被用于记载华夏-东夷共同语，也就是原始-上古汉语。这类似于现代日语的训读体系，即同一个概念/甲骨文字可能有两套读音，一套是东夷语固有的训读，另一套是华夏语的音读。甲骨文很大一部分（形声字）可能与已经消失的纯东夷语成分有关，因此难以释读。（2020-05-29）

## 上古汉语的m前缀

2024-12-29

说到“沐猴”“马猴”“蚂蚁”“蚂蚱”“吗喽”这些动物名称的m-前缀，据说是从原始汉藏语流传下来的，表示“有生命之物”的屈折标记，后来有些被保留下来。

另外一个线索是“麥”和“來”，有资料指出，小麦传入中国，最晚大概是在公元前2000年左右，对应王朝时代的夏朝。而小麦这种谷物在上古汉语的发音，大概是mrxx，表示的意思是“外面传来的（天赐的）某种有生命的（粮食作物）”。其中的m-前缀是表示“有生命之物”的屈折前缀，而rxx是抽象意义的“外来”“天上掉下来”的意思。

后来，创造了对应的符号来表示这种谷物，也就是“來”。至于“麥”，是“來”的异体字。借用象形的“來”或“麥”字，同时表示抽象意义的“来”和小麦的意思。秦汉后复辅音消失，“来rxx”和“麦mxx”两个词分化，分别用“來”和“麥”表示，來和麥这两个字也就有了现在通行的含义。

> 地质学家翁文灏在上世纪二十年代便注意到，当时地质时代译名，我国异乱纷呈而日本较统一，日本译名注意与前人保持一致，我国译者各译各的。他提出翻译科学名词要从先从众，“就习用之普遍言，则日译名词，既用汉文，即可仍用”，“沿用日名困难之点，唯在音译诸名。例如寒武志留，于彼国则恰如西音，在吾国则有同悬造……愚则谓从地诸名，原无深意，但便称呼，或用英地，或用日音，等是外物，何分彼此”。寒武，日语かんぶ（Kanbu），对应英文Cambrian；志留，しろ（Shiro），对应英文Silurian。实际上，非独地质，更有其他许多科技术语来自日文，经民国统一后，沿用至今，翻开上海出的《**汉语外来词词典**》便一目了然。我们现在用的汉语中，至少有一万个词是外来语。学习日语二十年的南京大学文学院教授王彬彬曾撰文说，我们今天使用的社会和人文科学方面的名词术语，有百分之七十是从日本输入的，这些都是日本人对西方相应语的翻译，传入中国后，便在汉语中牢牢扎根。“实际上，离开了日语‘外来语’，我们今天几乎就无法说话。”我今天才知道，非独社会人文社科，而更有许多科技术语，也来自日本。甚至“科学”这个词，也是日语。李兆忠在《汉字的圈套》中提出一个问题：同样一个西方科学术语，到了中国和日本，就被译成两种味道相去甚远的词，最后却是以中译失败，日译胜利而告终，这是为什么？我也不知道是为什么。可能要回答这个问题，就必须回答其他一些问题，但那些问题又是无法回答的。我又翻出去年出版的《四十六亿年的奇迹：地球简史》来看，这是我读到的最好的地理地质科普书，全书十几册都译自日文，自然，它里面又提到了寒武、志留这些名字。寒武、奥陶、志留基本都是英文词近代日译为日文汉字再原样传入中国。最搞的是德文郡（Devon）翻译成汉字“泥盆”（debon）。（综合 @韩松、@蘸盐 微博）

**女性名称**

**有实意的女名**

- 音读的：愛（あい）、蘭（らん）、凜（りん）；
- 单音拍训读的：江/枝（え）、音（ね）、葉（は）、帆/穂（ほ）；
- 双音拍训读的：
-- 彩/綾（あや）、琴（こと）、咲（さき）、月（つき）、花（はな）、舞（まい）、桃（もも）、雪（ゆき）、百合（ゆり）等；
-- 四季：春（はる）、夏（なつ）、秋（あき）、冬（ふゆ）；
- 三音拍训读的：
-- 葵（あおい）、泉（いずみ）、楓（かえで）、桜（さくら）、栞（しおり，书签的意思）、遥（はるか）、緑（みどり）、恵（めぐみ）、歩（あゆみ）、望（のぞみ）、光（ひかり）等。

**带后缀的女名**

- 音读的：敬子（けいこ）、優子（ゆうこ）、洋子（ようこ）；
- 训读的：和子（かずこ）、直子（なおこ）、紀子（のりこ）。

除了「子」以外，「美」（み）、「恵」（え）也是两个常见的后缀。它们前面一般只接双音拍训读汉字，一般也是跟男名共用的那些。比如：良美（よしみ）、広恵（ひろえ）。

- 长音、短音有时会混同，比如「英」「桜」「有」「礼、怜、玲、麗」等字本来都是长音。
- 「穂」字本来读 ほ，也可以用来代表 お 是因为ハ行転呼。
- 有些训读的字，比如「織」，既可以表示 おり 两个音拍，也可以只表示其中的 お，而 り 再单独配一个汉字。类似地，「結」可以表示 ゆい，也可以只表示其中的 ゆ。

一旦打开了「万叶假名」这个潘多拉魔盒，就出现了许多只顾音、不顾意的女名：先选定一个比较好听的读音，再从上面的「万叶假名」中随便选几个汉字。比如 かな 可以写作「加奈」、「佳南」、「香菜」，みき 可以写作「美紀」、「未樹」，まりこ 可以写作「麻里子」、「真理子」等等。有时，在挑选汉字的时候，也会兼顾其含义，比如「真理子」。穿越电影「東京少女」的女主角叫做 みほ，汉字写作「未歩」，根据剧情，赋予了「未来に歩く」（漫步未来）的含义。

**实例分析**

《名侦探柯南》历代主题曲、插曲的女歌手，分析一下她们的名字。

- 小松未歩（こまつ みほ）：属于万叶假名式，「未来に歩く」的含义不见得每个爸妈在给孩子起名时都会考虑。
- 坂井泉水（さかい いずみ）：属于有实意的类型。本来「いずみ」可以写成单字「泉」，这里加了一个「水」字，有一点儿万叶假名的意思（「水」读 みず）。
- 宇徳敬子（うとく けいこ）： 中规中矩的以「子」结尾的名字，「敬」字为音读。
- 中村由利（なかむら ゆり）：有实意 + 万叶假名式，「ゆり」本义为「百合」。
- 愛内里菜（あいうち りな）：万叶假名式，「りな」无实意。
- 倉木麻衣（くらき まい）：有实意 + 万叶假名式，「まい」本义为「舞」。
- 三枝夕夏（さえぐさ ゆうか）：万叶假名式，但音义兼备。
- 上木彩矢（かみき あや）：有实意 + 万叶假名式。「あや」本可以写成单字「彩」，这里又把其中的「や」配上了「矢」字。
- 滴草由実（しずくさ ゆみ）：万叶假名式，「ゆみ」本义为「弓」，但应该主要取音。
- 竹井詩織里（たけい しおり）：有实意 + 万叶假名式。「しおり」本义为「书签」，也常写作「詩織」，算是音义兼备，这里又给「り」单独配上了「里」字。
- 岩田さゆり（いわた さゆり）：写作假名，但本义是「小百合」。
- 上原あずみ（うえはら あずみ）：写作假名，无实义；如果一定要写成汉字，可以写作「杏美」（「杏」读 あず 来自「杏子」的唐音 あんず）。

## 科里亚克语和尼夫赫语

- 有人认为，堪察加地区的[科里亚克语](https://zh.wikipedia.org/wiki/%E7%A7%91%E9%87%8C%E4%BA%9A%E5%85%8B%E8%AF%AD)（Koryak）与朝鲜语（Korean）同源。据维基百科：字根kor解驯鹿。
- 据维基百科：[尼夫赫语](https://zh.wikipedia.org/wiki/%E5%B0%BC%E5%A4%AB%E8%B5%AB%E8%AA%9E)是一种孤立语言，有时归类于古西伯利亚语，其许多词汇与阿伊奴语、朝鲜语和阿尔泰语系的语言相似，但并没有规则的语音对应关系，因此这些相似的词汇很可能是借词。一说尼夫赫语与新罗语、伽倻语相似。
- 据维基百科：根据现有的考古资料，尼夫赫人的起源难以查考，他们传统的生活方式与堪察加半岛的堪察加人和科里亚克族很相似。
- 有人认为，科里亚克人和尼夫赫人被不少人类学家认为是古代濊貊——扶余人的后裔。据维基百科：濊人的文化是西团山文化，有人说是古亚细亚人科里亚克族有关。

## 关于扶余和濊貊的一篇小文章

> 来源：https://www.toutiao.com/i6408329391700443650/

> 秽貊为中国北方少数民族，因依秽水而居，故得此名。秽水，又做濊水，在今辽宁省凤城以东。两汉时期，秽貊族各部落得到了空前的发展，形成了许多部落联盟，并在此基础上建立了几个强大的地方民族政权。其中，居住在松嫩流域并逐渐向东南迁移的槁离国发展得最快，它和松花江大曲折处以南的秽国最早在史书中出现。

> 公元前2世纪，秽貊族的一支在嫩江流域建立了槁离国。在古文献中槁离国又称之为高夷，也就是后来的高句丽。有一年槁离国发生了内讧，王子东明出走，自乌裕尔河向南逃往秽国旧地。后来，在松花江大曲折以南建立了夫余国，历史上称作北夫余。关于北夫余建国的年代，史籍无确切记载。但从公元前108年（汉元封三年）置乐浪、临屯、玄菟、真番4郡后，在《史记》中开始有“原旧燕之地”，“北邻乌桓、夫余、东绾、秽貊、朝鲜、真番之利”的记载，说明夫余国至少建立于公元前108年以前。在秽貊族中夫余立国较早，发展较快，并且和高句丽、东夫余、百济等部族有着共同的渊源。夫余国的都城前期在今吉林市郊南城子，后期移至今吉林省农安镇。夫余国自汉武帝时即与汉朝建立了臣属关系，并接受西汉授予的印绶。西汉至两晋时期先后隶属于玄菟郡、辽东郡、公孙氏政权、曹魏、西晋、前燕和前秦等，长期朝贡于中原各代王朝。夫余国的疆域西至洮儿河，东到牡丹江，南至辉发河以北，北至黑龙江。据史料记载，夫余国始祖叫东明，建国后曾兴盛一时，到汉朝时夫余国发展进入鼎盛时期。到三国时期，夫余国占有松花江上游的广大地区，辖境包括今吉林中部和西部，“方可二千里，户八万”。“其民土著”，“其人强勇”，“其国殷富”，是一个经济、文化都相当发达的奴隶制政权。三国时曾协助曹魏打败高句丽。

> 在夫余存在的600年间，开拓了松嫩平原，创造了比较发达的文化，史称北夫余，是汉、魏、晋、南北朝时期东北的地方政权。是东北史上一个占有重要地位的古老的少数民族。

> 夫余之名始见《史记》，夫余又作凫臾、扶余或符。《尔雅》有“九夷之五为凫臾”，“凫臾，东方国名，即夫余也”。据《中国东北史》记载，夫余名称的起源有两种说法。一说来源于秽人之“秽”，如何秋涛谓，“符即夫余，《王会》之秽人也。”快读发音为“符”，如慢读则发音为“夫余”，秽即“夫余”二字的合音。另一说“夫余”二字来源于河流名，槁离国原来在乌裕尔河流域，该河在金代称蒲与，夫余即是蒲与或乌裕尔的同音异写。据上述两说，“秽”与“夫余”皆源于乌裕尔河之名。

# 上古歌谣

2011-12-16

上古时期的民歌、民谣，是民间文学的一种。我国古代，以合乐为歌，徒歌为谣。上古歌谣是先民表达思想、抒发感情、促进生产的重要工具，是在生产力极为低下的原始时代产生的，是出现最早的文学样式。按题材内容，可分为劳动歌谣、祭祀歌谣、图腾歌谣、婚恋歌谣、战争歌谣等。它们具有集体性、综合性和再现生活的直接性，词句简朴，节奏流畅，以赋为其主要表现手法。

原始歌谣大都采用二言形式，这是因为上古劳动动作简单，劳动节奏短促、鲜明，因而伴随劳动动作产生的诗歌节奏自然也不复杂。另外，上古汉语都是单音节词，两个单音节词组合是最初的句子，这种句子的产生与上古人的思维方式和语言能力直接相关。

上古歌谣零散保存在先秦两汉的典籍之中，后代集中辑本有：清代沈德潜的《古诗源》，今人逯钦立《先秦汉魏晋南北朝诗》收录最为详尽。但并不完全可信，有些如《击壤歌》、《卿云歌》、《南风歌》是出于后人的伪托。

**再现劳动过程**

《吴越春秋》中《弹歌》：断竹、续竹、飞土、逐宍。

相传为黄帝时代的作品，反映渔猎时代的劳动过程。弓箭的发明是人类摆脱蒙昧时代的重要标志。恩格斯说：“弓箭对于蒙昧时代，正如铁剑对于野蛮时代和火器对于文明时代一样，乃是决定性的武器”。（《家庭、私有制和国家的起源》）我国弓箭的发明很早，有所谓：“少昊生般，是始为弓”（《山海经·海内经》）；“羿作弓”。（《墨子·非儒》）其实弓箭的发明不是个人创造，而是原始人在漫长的时代中智慧和经验的积累。这首短歌流露着原始人对制造灵巧工具的自豪和喜悦，也表现了他们获取更多猎物的渴望。

**征服自然的愿望**

《礼记·郊特牲》中《蜡辞》：土反其宅，水归其壑，昆虫毋作，草木归其泽！

相传为伊耆氏时代的作品。伊耆氏，即神农氏，一说指帝尧。蜡，是古代一种祭礼的名称。周代在12月举行祭祀百神之礼，称为蜡礼，蜡礼上所用的祷辞，即称蜡辞。从这首短歌命令的口吻看，实际是对自然的“咒语”。大水泛滥，土地被淹没，昆虫成灾，草木荒芜，眼看收获无望，在原始宗教意识的支配下，原始人企图靠着这种有韵律的语言，来指挥自然，改变自然，使它服从自己的愿望。

**上古婚姻制度**

《周易》中《屯·六二》：屯如，邅如；乘马，班如；匪寇，婚媾。

这是一首抢婚的诗，一群男子骑在马上，迂回绕道而来，原以为是敌寇，等到闯进门来把姑娘抢走，才知道是为了婚事。反映了古代确实存在过的抢婚制度。诗短，但写得曲折形象，音韵和谐。

**战争**

《周易》中《中孚·六三》：得敌，或鼓、或罢、或泣、或歌。

这是一首写战争的诗，描写战争结束胜利归来的情景。战争胜利以后，有的仍在擂鼓示勇，有的坐卧休息，有的在哭泣，有的在引吭高歌。寥寥十字，写出了一个动人的场面。

**劳动生活**

《周易》中《归妹·上六》：女承筐，无实；士刲羊，无血。

牧场上男男女女们在剪羊毛、拾羊毛，男的剪羊毛，不见血；女的承筐装羊毛，不觉得有重量。轻快、生动，有情有景。

**思恋之歌**

《吕氏春秋·音初篇》中《候人歌》：候人兮猗。

传说大禹治水，娶涂山氏女为妻，大禹巡省南土，久不归，女乃唱了这首歌，渴望大禹归来。从历史发展的角度看，比较稳定的夫妻关系和夫妻感情，只有在一夫一妻制出现以后才有可能，而这时已属私有制萌芽后的氏族社会晚期。二字为句，语气词拖长尾音，取得了独特的抒情效果。这是中国最古老的情诗，开抒情诗传统之先河。
